# devConnect

## TUTORIAL 
1. `git clone <URL>`
2. `git checkout -b <Branch>`
3. **Kerjakan todo kalian ?**

## Setelah selesai mengerjakan 
1. `git add .` atau `git add <NAMA_FILE>`
2. `git commit -m "<MESSAGE>"`
3. `git push origin "<UR Branch>"` 
4. `Compare & Pull Request`, jangan di reviewers ke **Lead**. tunggu selesai di `Merge`.

## Kita akan pull dari github ke dalam local / laptop
1. pindah dulu ke branch `main`
2. lalu kita pull request dari github `git pull origin main`
